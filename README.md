# Hacker News Scraper (MERN Stack)

This is a full-stack web application built with the MERN stack (MongoDB, Express, React, Node.js). It scrapes top stories from Hacker News, stores them in a database, and provides a polished UI to view and bookmark them.

## Features
- Automated scraping of Hacker News top 10 stories.
- User Authentication (JWT based).
- Bookmark functionality.
- Premium UI built with React and Tailwind CSS.
